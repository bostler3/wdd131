Website Planning Document

Site Name: Harry Potter Fun

This name represents a site that provides fun Harry Potter-themed activities for all -- longtime fans and newbies alike!

Site Purpose

This site provides a "characters" page, a "sorting" quiz page that puts a user in a Hogwarts house, and a page that allows a user to send a message to a favorite Harry Potter character.

Scenarios

Questions a site user might ask:

  • Who are the main Harry Potter characters?
  • What Hogwarts house do I belong to?

Colors

Color ideas:

  • Header/Footer: background color is black and font color is a silver-gold gradient (#CFAB38)
  • Main: background color is a light gray (#F4F5F4) and font color is black
  • Other colors: where appropriate and color-contrast compliant, try to incorporate the Hogwart's house colors of red (#AE0001), green (#0E5B51), blue (#0B5A89) and yellow (#ECAF1F)

Typography

Font ideas:

  • Header/Footer:
    "Inkut Antiqua"
    from Google Fonts
  • Main: "Roboto" from Google Fonts
